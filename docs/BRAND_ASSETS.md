# Brand Assets

Ghostbrain uses `public/brand` as the Vite-served source directory for GUI identity assets.

## Current Final Sources

The current brand source set comes from the devdevbuilds current-finals export and should be treated as the canonical dashboard styling reference for this phase.

## Runtime Filenames

- `ghostbrain-primary.png`: Ghostbrain primary lockup.
- `ghostbrain-primary-wide.png`: Header-ready Ghostbrain primary lockup.
- `devdevbuilds-logo.png`: devdevbuilds studio logo.
- `devdevbuilds-wordmark.png`: Header-ready devdevbuilds studio mark.
- `ghostbrain-app-icon.png`: Circular app icon used by loading and About states.
- `favicon.png`: Browser favicon.
- `devdevbuilds-dashboard-bg.png`: Brushed-metal circuit dashboard background.
- `devdevbuilds-dashboard-hero.png`: Large branded dashboard plate.

## Additional Source Assets

- `devdevbuilds-icon-circle.png`: Circular metallic devdevbuilds icon.
- `devdevbuilds-square.png`: Square metallic devdevbuilds artwork.
- `devdevbuilds-domain.png`: devdevbuilds.com source card.
- `devdevbuilds-emails.png`: Studio contact source card.
- `devdevbuilds-email-embossed.png`: Single email source card.
- `devdevbuilds-circle-binary.gif`: Previous animated circular brand reference retained for future motion states.

Written studio references in code, docs, alt text, and configuration should use `devdevbuilds`. Source image artwork may preserve approved stylized lettering.
